Hiya, I'm Alex and welcome to the Orangey456 website! This is my personal website; a place where I can express myself and my interests, free from the cluster mess that is social media. A place online I can truly call home.

Some of the things I enjoy include photography, video production, vidya games, as well as tinkering with old and new technology. In our current modern tech world where websites are now more demanding and old computers struggle to keep up with them, I wanted to make this site work on older browsers and hardware just as well as the modern counterparts. This is inspired largely in part by Kugee's site Razorback, who really has set a good benchmark for how this can be done.

I also believe that old hardware can absolutely still be useful today for a lot of tasks. In fact, I'm writing this on a 15" PowerBook G4 from Late 2005 and have been using various other older computers in my collection to work on different aspects of this site.

Of course, this website is still currently a work in progress. It will be temporairly hosted on Github until I eventually figure out a hosting solution that would work best for me; which means only newer browsers that support TLS 1.3/SSL can access the site for now (unless you use some sort of HTTPS proxy).

Thank you so much for visiting and I hope you enjoy your stay! :)
